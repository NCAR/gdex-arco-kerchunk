# Kerchunk Reference Pipeline

This directory contains Python script examples demonstrating how to generate kerchunk reference file for various datasets. These scripts serve as practical guides for creating reference file using the tools provided in this repository.

## Overview

The examples show real-world usage patterns for the `create_kerchunk.py` tool and related utilities. Each script is tailored for specific datasets and demonstrates different configuration options and data access patterns.

## Available Scripts

- [`d640000.jra3q.py`](https://github.com/NCAR/gdex-arco-kerchunk/blob/main/examples/d640000.jra3q.py) : script to create the JRA3Q reference file that contain posix (GLADE access), https (https URL access), osdf (open science data federation access)
